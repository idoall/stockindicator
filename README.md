# 常用的股票指标实现

### 主要分为三类

 - [通道类技术指标](./channel/README.md)
 - [趋势类技术指标](./trend/README.md)
 - [交易量相关技术指标](./volume/README.md)
 - [震荡类技术指标](./oscillator/README.md)




多一个指标作参考及分析固然是好，但世上没有无敌指标的，并非用了那个指标进行交易便能必胜，要明白指标的原理及优点，将其融入你个人的交易策略做分析，看看是否能提高回报，这才是正确的做法!
